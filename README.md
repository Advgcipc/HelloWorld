Markdown支援有序清單和無序清單。

無序清單使用星號、加號或是減號作為清單標記：

*   Red
*   Green
*   Blue

等同於：

+   Red
+   Green
+   Blue

也等同於：

-   Red
-   Green
-   Blue

有序清單則使用數字接著一個英文句點：

1.  Bird
2.  McHale
3.  Parish
